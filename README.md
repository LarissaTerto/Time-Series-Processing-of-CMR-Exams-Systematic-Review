This repository was created as part of the Systematic Review titled: Time Series Processing of Cardiac Magnetic Ressonance Exams.
File Protocol.xlsx includes the review protocol and the data extraction for each of the selected studies;
file Graphs.pbix presents graphs to summarize the information extracted from the studies;
file RS_Larissa.start includes the full systematic review process created in the auxiliary tool Start (https://www.lapes.ufscar.br/resources/tools-1/start-1).
